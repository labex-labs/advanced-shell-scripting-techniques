高度なシェルスクリプティング技術を学びます。このモジュールでは、シェル関数の基本、変数のスコープとリターンステータス、Trap を使用したエラー処理、文字列操作技術、配列の処理について解説します。実践的なラボと実世界の課題を通じて、これらの不可欠な Linux スキルを習得します。
